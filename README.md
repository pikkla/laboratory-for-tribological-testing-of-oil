# laboratory-for-tribological-testing-of-oil
Plotting dependency graphs
The code allows you to plot graphs of the dependence of the friction moment and temperature on the pressing force.
 Source file format:
 0_0.txt - clamping force values ​​Fn [N]
 0_1.txt - temperature values ​​T [°C]
 0_2.txt - friction torque values ​​M [N m]
